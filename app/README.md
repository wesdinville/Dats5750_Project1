# Simple E-commerce App

This is a basic frontend + Node.js web app you can deploy on Google Compute Engine.

## To run locally

```bash
npm install
npm start
```

## Structure

- `index.html` - Product display and cart
- `style.css` - Styling
- `server.js` - Node server
- `package.json` - Node dependencies

## Deployment

Works on:
- Compute Engine
- Cloud Run
- Kubernetes
